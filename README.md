# android-playground
Androidアプリケーションのサンプルです。


## Gradle の設定

* BuildTypeによってandroid:labelを変更する（インストール時の表示名を変更する）。
* assemble時にBuildType毎に生成されるAPKファイルのファイル名を変更する。
* Dokkaの設定
* ユニットテストのカバレッジレポート出力
